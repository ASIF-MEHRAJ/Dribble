A high-fidelity, fully responsive front-end clone of the Dribbble homepage. This project focuses on modern web layouts using CSS Flexbox and CSS Grid to create a seamless user experience across mobile, tablet, and desktop viewports.

🚀 Live Preview
   https://asif-mehraj.github.io/Dribble/

✨ Features
Responsive Navigation: Adapts automatically from a desktop link tree to a clean, mobile-friendly layout utilizing a hamburger menu icon on smaller screens.

Fluid Search Bar: Built with dynamic constraints (max-width: 45rem and width: 100%) ensuring the search bar contracts gracefully on mobile screens without overflowing.

Adaptive Grid Matrix: The card gallery leverages CSS Grid with grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)) to let project cards dynamically wrap into rows based on screen width.

Interactive Image Overlays: Clean CSS transitions control sleek card hover states, revealing project titles, custom linear-gradient shadows, and action utilities.

Infinite Scrolling Marquee: Features a hardware-accelerated logo carousel using CSS @keyframes animations for a continuous corporate showcase section.

Remix Icon Integration: Scalable vector iconography throughout the interface to maintain visual clarity on high-density (Retina) displays.

🛠️ Tech Stack
HTML5: Semantic document structure using structural elements like <nav>, <main>, <section>, and <footer>.

CSS3: Custom styling driven by responsive modern architectures:

Flexbox for aligned layouts (Navigation, Search blocks, and Footers).
